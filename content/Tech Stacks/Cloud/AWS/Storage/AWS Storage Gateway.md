## AWS Storage Gateway
---
- Bridge between on-premises data and cloud data
- Use cases:
	- DR
	- backup & restore
	- tiered storage
	- on-premises cache & low-latency files access
- Types of Storage Gateway:
	- S3 File Gateway
	- FSx File Gateway
	- Volume Gateway
	- Tape Gateway
