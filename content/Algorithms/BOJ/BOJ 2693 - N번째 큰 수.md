## 풀이

---

파이썬 기본문법만 활용하면 쉽게 해결할 수 있는 문제이다.

주어진 배열을 내림차순으로 정렬 후 3번째 자리에 있는 숫자를 출력하면된다.

## 소스코드

---

```python
import sys
input = sys.stdin.readline

t = int(input())

for _ in range(t):
    a = sorted(list(map(int, input().split())), reverse=True)
    print(a[2])
```
