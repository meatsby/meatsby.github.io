# 7. 변수와 리터럴의 타입 불일치

- 범위가 ‘변수 > 리터럴’ 인 경우, OK
    
    ```java
    int i = 'A';      // int > char
    // 'A' 는 65
    long l = 123;     // long > int
    double d = 3.14f; // double > float
    ```
    
- 범위가 ‘변수 < 리터럴’ 인 경우, 에러
    
    ```java
    int i = 3_000_000_000; // int의 범위 벗어남
    long l = 3.14f;        // long < float
    float f = 3.14         // float < double
    ```
    
- byte, short 변수에 int 리터럴 저장 가능
    - 단, 변수의 타입의 범위 이내이어야 함
    
    ```java
    byte b = 100; // OK
    byte b = 128; // 에러, (-128 ~ 127) 범위 벗어남
    ```
