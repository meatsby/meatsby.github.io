# 5. 증감 연산자

> `증가 연산자(++)` 피연산자의 값을 1 증가시킨다.
> 

> `감소 연산자(--)` 피연산자의 값을 1 감소시킨다.
> 

| 타입 | 설명 | 사용 예 |
| --- | --- | --- |
| 전위형 | 값이 참조되기 전에 증가시킨다. | `j = ++i;` |
| 후위형 | 값이 참조된 후에 증가시킨다. | `j = i++;` |

```java
// 전위형
j = ++i;

// 아래와 같다
++i;   // 증가 후에
j = i; // 참조하여 대입

// 실행 이후
i = i
j = i + 1
```

```java
// 후위형
j = i++;

// 아래와 같다
j = i; // 참조하여 대입 후에
i++;   // 증가

// 실행 이후
j = i
i = i + 1
```
