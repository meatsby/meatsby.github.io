# 21. 접근 제어자

## 접근 제어자 (access modifier)

- `private` - 같은 클래스 내에서만 접근 가능
- `(default)` - 같은 패키지 내에서만 접근 가능
- `protected` - 같은 패지키 내에서, 그리고 다른 패키지의 자손클래스에서 접근 가능
- `public` - 접근 제한이 없음

| 제어자 | 같은 클래스 | 같은 패키지 | 자손 클래스 | 전체 |
| --- | --- | --- | --- | --- |
| `public` | ⭕ | ⭕ | ⭕ | ⭕ |
| `protected` | ⭕ | ⭕ | ⭕ |  |
| `(default)` | ⭕ | ⭕ |  |  |
| `private` | ⭕ |  |  |  |
