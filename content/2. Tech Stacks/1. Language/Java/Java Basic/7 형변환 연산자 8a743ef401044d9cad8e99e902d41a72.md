# 7. 형변환 연산자

### 형변환이란,

> 변수 또는 상수의 타입을 다른 타입으로 변환하는 것
> 

```java
(타입)피연산자

double d = 85.4;
int score = (int)d;
```

| 변환 | 수식 | 결과 |
| --- | --- | --- |
| int → char | `(char)65` | `'A'` |
| char → int | `(int)'A'` | `65` |
| float → int | `(int)1.6f` | `1` |
| int → float | `(float)10` | `10.0f` |
