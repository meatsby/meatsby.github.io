# 12. 형식화된 출력 - printf()

- println()의 단점 - 출력형식 지정불가
    - 실수의 자리수 조절불가 - 소수점 n자리만 출력하려면?
        
        ```java
        System.out.println(10.0/3); // 3.333333...
        ```
        
    - 10진수로만 출력된다
        
        ```java
        System.out.println(0x1A); // 26
        ```
        
- printf()로 출력형식 지정가능
    
    ```java
    System.out.printf("%.2f", 10.0/3); // 3.33
    System.out.printf("%d", 0x1A); // 26
    System.out.printf("%x", 0x1A); // 1A
    ```
