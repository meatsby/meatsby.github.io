# 18~20. 2차원 배열

### 18. 2차원 배열

> 테이블 형태의 데이터를 저장하기 위한 배열
> 

```java
int[][] score = new int[4][3]; // 4행 3열의 2차원 배열을 생성
```

### 20. 2차원 배열의 초기화

```java
int[][] score = new int[][]{ {1, 2, 3}, {4, 5, 6} };
int[][] score = { {1, 2, 3}, {4, 5, 6} }; // new int[][] 생략

// 직관적으로 초기화
int[][] score = new int[][]{
															{1, 2, 3},
															{4, 5, 6}
														};
```
