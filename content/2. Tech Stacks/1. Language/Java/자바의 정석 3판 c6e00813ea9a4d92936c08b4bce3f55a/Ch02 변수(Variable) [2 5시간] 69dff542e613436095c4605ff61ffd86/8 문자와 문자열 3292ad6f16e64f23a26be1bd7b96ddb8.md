# 8. 문자와 문자열

```java
char ch = 'A';
char ch = 'AB'; // 에러
String s = "ABC";

// String은 클래스
String s1 = "AB";             // s1에 주소를 저장
String s2 = new String("AB"); // 생성자 생략 가능

String s = "A";
String s = ""; // 빈 문자열
char ch = '';  // 에러

String s1 = "A" + "B"; // "AB"
""+7 -> ""+"7" -> "7"
""+7+7 -> "7"+7 -> "7"+"7" -> "77"
7+7+"" -> 14+"" -> "14"+"" -> "14"
```
