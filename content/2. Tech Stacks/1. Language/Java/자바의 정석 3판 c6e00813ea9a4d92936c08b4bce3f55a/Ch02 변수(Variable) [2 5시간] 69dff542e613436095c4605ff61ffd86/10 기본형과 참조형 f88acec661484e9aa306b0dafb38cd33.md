# 10. 기본형과 참조형

![Untitled](10%20%E1%84%80%E1%85%B5%E1%84%87%E1%85%A9%E1%86%AB%E1%84%92%E1%85%A7%E1%86%BC%E1%84%80%E1%85%AA%20%E1%84%8E%E1%85%A1%E1%86%B7%E1%84%8C%E1%85%A9%E1%84%92%E1%85%A7%E1%86%BC%20f88acec661484e9aa306b0dafb38cd33/Untitled.png)

- 기본형(Primitive Type)
    - 오직 8개 (boolean, char, byte, short, int, long, float, double)
    - 실제 값을 저장
    - 각 타입마다 메모리 사용 크기가 정해져 있기 때문에 범위를 초과할 경우 오버플로우 현상이 발생할 수 있음
    - 스택 영역에 생성
- 참조형(Reference Type)
    - 기본형을 제외한 나머지 (String, System 등)
    - 메모리 주소를 저장(4 byte 또는 8 byte)
    - null 값을 대입할 수 있다면 참조 타입
    - 힙 영역에 생성
    
    ```java
    Date today; // 참조형 변수 today를 선언
    today = new Date(); // 객체생성 후 today에 객체의 주소를 저장
    ```
